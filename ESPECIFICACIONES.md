# Especificaciones Técnicas - backVR Architecture

## 1. Objetivo
Desarrollo de un sistema de navegación VR interactivo centrado en hotspots para visualización arquitectónica.

## 2. Componentes Principales
- **HotspotManager.ts**: Clase principal que gestiona la lógica de los puntos de interacción, detección de clics y disparadores de eventos en la escena.
- **Configuración**: El sistema está preparado para integrarse con modelos 3D y entornos 360°.
